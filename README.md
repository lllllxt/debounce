# debounce

防抖函数，懂的都懂

## 安装

```
npm i @lllllxt/debounce
```

## 使用

`debounce(fn, [delay=500], [immediate=false])`

| 参数      | 类型/默认      | 描述                 |
| --------- | -------------- | -------------------- |
| fn        | function       | 要防抖动的函数       |
| delay     | number，500    | 需要延迟的毫秒数     |
| immediate | boolean，false | 是否立即执行 fn 函数 |
